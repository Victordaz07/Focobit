# Focobit

App multiplataforma para personas con TDAH.

## Stack

- **Mobile:** Expo (React Native) con TypeScript
- **Web:** Next.js 14 con TypeScript
- **Backend:** Firebase (Firestore + Auth + Functions)
- **Monorepo:** pnpm workspaces

## Comandos

```bash
# Instalar dependencias
pnpm install

# Mobile (Expo)
pnpm mobile

# Web (Next.js)
pnpm web

# Firebase Functions (watch)
pnpm functions

# Build
pnpm build:web
pnpm build:functions

# Type check
pnpm type-check
```
